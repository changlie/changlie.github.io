http server

HttpServer

HttpServer是内置函数httpServer(post)的返回値类型

方法 描述
get(path, func) 定义一个get 服务
post(path, func) 定义一个post 服务
startup() 启动http服务

get 服务

server = httpServer(8888)

server.get("/hi", $ req, resp {
    resp.txt("Hello World!")
})

server.startup()

post 服务

server = httpServer(8888)

server.post("/newUser", $ req, resp {
    // 打印参数
    req.args().pretty()

    resp.json({"result":"ok"})
})

server.startup()
